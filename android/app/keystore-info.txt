IMPORTANT SECURITY NOTES:
-------------------------
1. Keep the keystore file (upload-keystore.jks) secure and backed up
2. NEVER commit the keystore or this info file to version control
3. Change the default passwords to something more secure
4. Store credentials in a secure password manager

For Play Store Upload Key Change:
----------------------------------
1. Go to Google Play Console
2. Navigate to: Setup > App signing
3. Request upload key reset
4. Upload the file: upload_certificate.pem

To update your build.gradle:
-----------------------------
Add the following to android/app/build.gradle in the android block:

    signingConfigs {
        release {
            storeFile file('upload-keystore.jks')
            storePassword 'android123'
            keyAlias 'upload'
            keyPassword 'android123'
        }
    }
    
    buildTypes {
        release {
            signingConfig signingConfigs.release
            // ... other release config
        }
    }
